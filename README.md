# MicroEndo

**Integrating the Endometrial Microbiome & Single-Cell Niche in Asherman's Syndrome**

## Overview
Asherman’s Syndrome (AS) is an acquired endometrial disorder characterized by fibrotic intrauterine adhesions and reproductive failure. This repository houses the code to perform multi-omics integration of the host's single-cell RNA-sequencing data alongside metagenomic microbiome profiles to uncover biomarkers of severity and therapeutic success.

## Project Structure
- `docs/`: Technical specifications, design documents, and the active task backlog.
- `data/`: Directory for large biological datasets (ignored in git).
- `src/`: Source code for analytical models.
- `results/`: Output directories for generated plots and reports.

## Getting Started
Ensure all datasets are placed inside the `data/` directory. Check `GEMINI.md` for specific contextual instructions and conventions.
