# Product Requirements Document (PRD)

## Vision
The MicroEndo project aims to deliver a robust, reproducible analytical pipeline that uncovers the biological interplay between the endometrial host niche and the resident microbiome in Asherman's Syndrome.

## Scope
1. **Data Preprocessing**: Aggregate scRNA-seq matrices into pseudobulks.
2. **Correlation Mapping**: Identify univariate interactions between specific microbes (e.g. *L. iners*, *G. vaginalis*) and host inflammatory responses.
3. **Unsupervised Discovery**: Uncover latent biological factors using MOFA+.
4. **Supervised Discovery**: Uncover predictive clinical biomarkers using mixOmics.
